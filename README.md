# 🎸 Bossle

**A Bruce Springsteen Heardle — made with love for Father's Day.**

Think you know The Boss? Bossle gives you a tiny clip of a Springsteen classic and challenges you to name it before the clock runs out. Each wrong guess or skip unlocks a longer listen — but real fans won't need more than a second.

---

## How to Play

1. Hit **▶** to play the opening clip of a mystery Springsteen song.
2. Search for the song title and **Submit Guess**, or **Skip** to hear more.
3. Each failed attempt reveals a longer clip:

   `1s → 2s → 4s → 8s → 16s → 31s`

4. You get **6 attempts**. Fewest guesses wins bragging rights.
5. After each round, hit **▶ NEW SONG** to play again — unlimited rounds!

---

## Songs

The catalog spans Bruce's career from *Greetings from Asbury Park, N.J.* all the way through his later work — Born to Run, Darkness on the Edge of Town, Nebraska, Born in the U.S.A., and more. If you've been listening since the '70s, you've got a serious edge.

## Tech

Built with vanilla HTML, CSS, and JavaScript — no frameworks, no fuss.

- `index.html` — structure and layout
- `style.css` — styling (dark asphalt palette, jersey red and amber gold)
- `game.js` — game logic, audio playback, waveform animation
- `songs.js` — the full Springsteen song catalog
- `audio/` — song clips

**Claude (Anthropic's AI) was used to help build and debug this project** — including the waveform animation logic, audio playback timing, and autocomplete search.
